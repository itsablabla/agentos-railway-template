# Research Skill

A structured methodology for conducting deep, comprehensive research on any topic.

## When to Use This Skill

Use this skill when you need to investigate a topic thoroughly from multiple angles,
produce a well-sourced research report, fact-check claims with primary sources,
or synthesize information from diverse sources.

## Research Framework

### Phase 1: Scoping
Define the core research question clearly. Identify 3-5 sub-questions that together
answer the main question. Determine the required depth and note any constraints.

### Phase 2: Source Strategy
Use primary sources (official documents, original studies, direct data),
secondary sources (news articles, analysis, commentary),
academic sources (arXiv, PubMed), and community sources (HackerNews, Reddit).

### Phase 3: Information Gathering
1. Start broad — get an overview from Wikipedia and DuckDuckGo
2. Go deep — find primary sources for key claims
3. Find contrarian views — actively search for opposing perspectives
4. Check recency — ensure information is current

### Phase 4: Synthesis
Identify consensus views vs. contested claims. Note confidence levels for each finding.
Highlight gaps in available information. Draw connections between disparate findings.

### Phase 5: Output Structure
- **Executive Summary**: 3-5 sentence overview
- **Key Findings**: Numbered list with source citations
- **Analysis**: Synthesis and interpretation
- **Uncertainties**: What we don't know
- **Sources**: Full citation list

## Quality Checklist
- At least 3 independent sources for major claims
- Recency verified (check publication dates)
- Contrarian views considered
- Confidence levels noted
- All claims linked to sources
