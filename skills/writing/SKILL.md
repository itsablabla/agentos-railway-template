# Writing Skill

A framework for producing clear, compelling, and effective written content.

## When to Use This Skill

Use this skill when writing blog posts, articles, or long-form content;
drafting professional emails or communications; creating documentation or guides;
editing and improving existing text; or writing executive summaries or reports.

## Writing Principles

### The Pyramid Principle
Lead with the conclusion, then support it:
1. Answer first: State the main point immediately
2. Support: Provide 3-5 supporting arguments
3. Detail: Elaborate on each argument

### Clarity Rules
- Use active voice: "We built X" not "X was built by us"
- Short sentences: Average 15-20 words
- Plain language: Avoid jargon unless writing for experts
- Concrete specifics: "3x faster" not "significantly faster"
- One idea per paragraph

### Structure Templates

**Executive Summary:**
- What: One sentence describing the situation or decision
- So what: Why it matters (impact)
- Now what: Recommended action
- Details: Supporting evidence (optional)

**Technical Documentation:**
- Title and brief description of what this does and when to use it
- Quick Start: Minimal working example
- Usage: Detailed usage with examples
- Reference: Parameters, options, return values
- Troubleshooting: Common issues and solutions

**Professional Email:**
- Subject: [Action] + [Topic] (e.g., "Approve: Q2 Budget Proposal")
- Context: One sentence of background
- Ask: Specific request or action needed
- Details: Supporting information
- Deadline: If applicable

## Editing Checklist
- Is the main point clear in the first paragraph?
- Does each paragraph have one clear idea?
- Are there unnecessary words? (cut 20%)
- Is the tone appropriate for the audience?
- Are all claims supported?
- Is the call-to-action clear?
